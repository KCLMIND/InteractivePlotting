This is a simple streamlit / python based plotter of data. 

It fits a linear / quadratic / cubic fit to the main data, estimates the residuals, fits a line to that, and from there you get a quick growth chart. 

The options are fairly self explanatory. As there are lots of outliers in the imaging data, there is an option to fit the curve, find subjects who fall outside the normal range (set by you, Z=4 seems fine) and fit again.
